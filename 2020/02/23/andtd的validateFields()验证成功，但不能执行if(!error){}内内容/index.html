问题：点提交按钮后执行validateFields()验证成功，但不能执行if(!error){//验证成功}内内容

分析及解决：

问题产生原因是： 使用了自定义验证函数validator:valiPrice，它的每个分支必须执行callback()函数，否则将无法通过验证
【1】此处必须进行回调函数调用，否则【2】处将无法通过验证并执行

其它写法：直接把验证函数写在里面
