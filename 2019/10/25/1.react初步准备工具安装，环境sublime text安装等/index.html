sublime Text安装汉化插件

  1. ctrl+`，控制台输入如下，回车
  2. 然后点setting-default，菜单在：
    【preferences——package settings——package control——settings-default】
    把此网址注释掉，改成.cn的网址，提高安装速度
  3. 再【preferences——package control】，找到里面的：移上去，回车
  4. 再输入chinese….找到【Chinese​Localizations】移上去，回车。等等完成安装
  5. 再菜单里【help——language——找到中文】选中，软件即变成汉化版
